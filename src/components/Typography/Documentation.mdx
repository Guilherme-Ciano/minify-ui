# Typography Component Documentation

The `Typography` component provides a set of typographic elements such as headings, paragraphs, and links with customizable styles including font weight, font style (italic), and text decoration (underline). These components offer flexibility in styling text content across the application.

## Examples

These examples demonstrate different use cases and configurations of the `Typography` components, providing a comprehensive set of typographic elements for text display and interaction within the application.

```tsx
import { H1, H2, H3, P, A, Small } from 'minify-ui';

// Heading elements
<H1 type="normal">Heading 1</H1>
<H2 type="bold">Heading 2</H2>
<H3 type="light">Heading 3</H3>

// Paragraph element
<P type="normal">Lorem ipsum dolor sit amet, consectetur adipiscing elit.</P>

// Link element
<A href="#" type="bold">Click here</A>

<Small>Small text</Small>
```

## Props

The `Typography` components accept the following props:

### Common Props

- `type`: Specifies the text style variant. Options include `'light'`, `'lightBold'`, `'normal'`, `'bold'`, and `'bolder'`. Default is `'normal'`.
- `italic`: Specifies whether the text should be italicized. Default is `false`.
- `underline`: Specifies whether the text should be underlined. Default is `false`.

### Heading Props

- `children`: Text content to be displayed within the heading element.

### Paragraph Props

- `children`: Text content to be displayed within the paragraph element.

### Link Props

- `children`: Text content or JSX content to be displayed within the link element.
- `href`: Specifies the URL of the link.
- Other anchor HTML attributes like `target`, `rel`, etc.
